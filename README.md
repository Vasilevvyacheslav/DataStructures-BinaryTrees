# Binary Search Trees Lab (BinaryTreeLab)

Лабораторная работа по структурам данных: реализация **Бинарного Дерева Поиска (BST)** с алгоритмами балансировки (AVL), обхода и пошаговой визуализации.

### Вариант №24: Тип данных double, поиск максимального элемента

## Возможности

- **Бинарное Дерево Поиска**: Вставка, удаление, поиск узлов (тип `double`).
- **Балансировка**: Автоматическая перестройка дерева для минимальной высоты.
- **Обходы**: Прямой (Preorder), Симметричный (Inorder), Обратный (Postorder).
- **Визуализация**: Графическое отображение структуры дерева через `QGraphicsScene`.
- **Анимация**: Пошаговая подсветка узлов при обходе и поиске (жёлтый — текущий, зелёный — пройденный).
- **Печать**: Вывод дерева в консоль (вертикально по уровням и горизонтально с поворотом).

## 🛠 Технологии

- **Язык**: C++17
- **Фреймворк**: Qt 6 (Widgets, QGraphicsView)
- **Сборка**: CMake
- **Графика**: `QGraphicsScene` для отрисовки узлов и связей
